# 安裝
安裝[Python 3.10.6](https://www.python.org/downloads/release/python-3106/) 

下載這個資料庫 

在文件路徑上執行cmd
並執行下方命令

```cmd
pip install virtualenv
virtualenv venv
venv\Scripts\activate.bat
pip install -r requirements.txt

```

以系統管理員身分執行 LoRA extraction only
